## Sobre o repositório

Esse repositório reune todos os exercícios de Java desenvolvidos dentro das aulas das disciplinas de **Programação Orientada a Objeto** e **Laboratório de Programação Orientada a Objetos**, do curso de Engenharia da Computação da Universidade de Ribeirão Preto (UNAERP).

Os exercícios estão dividos em duas pastas principais: ```Fundamentos_POO``` (com os exercícios da disciplina de Programação Orientada a Objeto) e ```Interfaces Gráficas``` (com os exercícios da disciplina de Laboratório de Programação Orientada a Objetos).
